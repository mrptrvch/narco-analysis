# Drug Overdose Analyzer

Программа скачивает данные о смертности от передозировок наркотиками из Our World in Data,
суммирует ставки по типам веществ и сохраняет результаты в JSON.

## Запуск
python src/main.py

#### загруженные данные сохраняются в data/raw, обработанные данные - в json файл data/processed/
